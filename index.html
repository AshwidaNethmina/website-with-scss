Goals of Data Security

  1. Confientiality

    2. Assets can be only accessed by authorized people

  2. Intergrity

    3. Assets can be only changed by authorized people

  3. Availabiliti

    4. Assets can be accessed when required

  4. Authentication

    5. Ability of system to can confirm the identity of a sender

  5. Non-redudiation

    6. Ability of system to can confirm that a sender cannot convincingly deny doing something
